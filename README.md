# Django AI Info Site

An informational website about Artificial Intelligence, built with Django. Includes a blog, FAQ, and pricing pages.

🌐 **Live Demo:** coming soon

---

## Features

- **Blog** — Articles and posts about AI topics
- **FAQ Page** — Frequently asked questions about AI
- **Pricing Page** — Plans and pricing overview
- **User Authentication** — Register and login system
- **Admin Panel** — Django admin for content management
- **Responsive Design** — Clean layout with CSS

---

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Backend | Python · Django |
| Frontend | HTML · CSS · JavaScript |

## Getting Started

```bash
# Clone the repository
git clone https://github.com/Afsaneh-hassani/django-ai-info-site.git
cd django-ai-info-site

# Create a virtual environment
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Apply migrations
python manage.py migrate

# Create a superuser
python manage.py createsuperuser

# Run the development server
python manage.py runserver
```

Then visit `http://127.0.0.1:8000` in your browser.
